# Momentum Todo App

간단한 할 일 관리와 생산성 향상을 위한 Flutter 앱입니다.

## 주요 기능

- 할 일 추가, 편집, 삭제 기능
- 오늘, 내일, 이번 주 할 일 관리
- 월별 목표 설정 및 추적
- 깔끔하고 직관적인 UI/UX

## 시작하기

이 프로젝트를 실행하기 위해서는 Flutter 개발 환경이 필요합니다.

```bash
# 의존성 설치
flutter pub get

# 앱 실행
flutter run
```

## 기술 스택

- Flutter
- Provider 상태 관리
- 로컬 저장소(SharedPreferences)
